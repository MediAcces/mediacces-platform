# MédiAccès - Back-Office Web

Interface web pour l'administration, les hôpitaux et les pharmacies.

## Stack prévue

- React.js
- React Router
- Axios
- Tailwind CSS

## Modules prévus

### Administration
- Dashboard avec vue d'ensemble
- Gestion des utilisateurs (validation chauffeurs/coursiers)
- Gestion des missions en temps réel
- Gestion financière et commissions
- Paramétrage des tarifs

### Hôpitaux / Cliniques
- Réservation centralisée de transport
- Annuaire patients récurrents
- Facturation groupée
- Historique et rapports

### Pharmacies
- Gestion des commandes de livraison
- Annuaire patients
- Suivi des livraisons en temps réel
- Facturation

## Implémentation

Ce module sera développé lors des Sprints 9 à 14. La structure sera initialisée avec Create React App ou Vite lors de cette phase.
