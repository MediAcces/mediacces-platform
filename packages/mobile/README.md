# MédiAccès - Application Mobile

Application mobile pour les patients et les prestataires (chauffeurs / coursiers).

## Stack prévue

- React Native avec Expo
- React Navigation
- Axios
- AsyncStorage
- React Native Maps

## Modules prévus

### Application Patient
- Inscription / Connexion (OTP, email, réseaux sociaux)
- Recherche de transport / livraison de médicaments
- Affichage des prestataires disponibles
- Réservation et suivi en temps réel
- Paiement (Mobile Money, carte bancaire, espèces)
- Historique et factures
- Notifications push et SMS
- Support client (chat intégré)

### Application Chauffeur / Coursier
- Inscription avec soumission de documents
- Gestion de disponibilité (en ligne / hors ligne)
- Réception et gestion des demandes
- Navigation intégrée
- Gestion des gains et historique
- Communication anonymisée

## Implémentation

Ce module sera développé lors des Sprints 3 à 6. L'application sera initialisée avec Expo lors de cette phase.

## Compatibilité

- Android 6+
- iOS 13+
